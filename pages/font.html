What are Font and Color Tags?

The font and color tags are older HTML tags that were used to apply font styles and text colors to elements on a web page. However, they are considered deprecated and are no longer recommended for use in modern HTML. Instead, CSS (Cascading Style Sheets) is the preferred method for styling content in HTML.
Let's briefly discuss how these tags were used and then cover the recommended way to achieve similar effects using CSS.

1. The Font Tag The font tag was used to apply various font-related styles to text within the tag. You could specify attributes like face for the font family and size for the font size.

Example font face="Arial" size="4" color="red"> This is some text.

2. The Color Tag The color tag was used to change the text color.

Example: color="blue" This text is blue.

How to Use CSS for Font and Color Styling:

CSS is the preferred method for styling content in HTML. The following example shows how to use CSS to style text.

Example:

<style> p {
    font-family: Arial;
    font-size: 16px;
    color: red;
}
</style>

Common CSS Properties for Text Styling

The following table lists some of the most common CSS properties used to style text.


Attribute Function
font-family Specifies the font family for text.
font-size Specifies the size of the font.
color Defines the color of the text.
background-color Sets the background color of an element.
font-weight Controls the boldness of text (e.g., normal, bold).
font-style Specifies the style of text (e.g., normal, italic).
text-decoration Adds decoration to text (e.g., underline, strikethrough).
letter-spacing Adjusts the space between characters in text.

Ordered List: Fonts and Colors in HTML/CSS

  1. Font Properties
    1. font-family: Specifies the font family for text.
    2. font-size: Defines the size of the font.
    3. font-weight: Controls the boldness of text (e.g., normal, bold).
    4. font-style: Specifies the style of text (e.g., normal, italic).
    5. text-decoration: Adds decoration to text (e.g., underline, strikethrough).
    6. letter-spacing: Adjusts the space between characters in text.
  2. Color Properties
    1. color: Defines the color of the text.
    2. background-color: Sets the background color of an element.
    3. border-color: Specifies the color of element borders.
    4. box-shadow: Adds a shadow to elements, including a color component.
    5. text-color: Deprecated; use color for text color.
