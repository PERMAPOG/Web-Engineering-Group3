The <img> tag


Brian Bao

Paragraph

The <img> tag in HTML is used to embed images in a web page.
It stands for "image" and is a self-closing tag, meaning it doesn't have a closing tag.
Instead, it's used with attributes to specify the source of the image and control various aspects of its display.

Table

Attributes What it does?
src This attribute specifies the URL or file path of the image you want to display.
It can be a relative path (relative to the HTML file) or an absolute URL.
alt This attribute provides alternative text for the image. It is important for accessibility purposes and should describe the content of the image.
If the image cannot be displayed for any reason, the alt text will be shown instead.
width This attribute sets the width of the image in pixels. You can also use percentages to make the image responsive.
height This attribute sets the height of the image in pixels. Like the width attribute, you can also use percentages.
title This attribute provides a title or tooltip that appears when the user hovers over the image.
align This attribute (now considered deprecated) used to control the alignment of the image within the text flow (e.g., "left," "right," "center").
border This attribute (also deprecated) used to add a border around the image.
style You can use inline CSS to apply specific styles to the image, such as margins, borders, and more.
class You can assign a class to the image, which can then be styled using external CSS.
