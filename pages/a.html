The <a> Tag

The HTML <a> tag is used to create hyperlinks that link one webpage to another, or to different sections within the same page. It stands for "anchor" and is a fundamental building block of the World Wide Web. The <a> tag is an inline element and is typically used within the content of other block-level elements like <div> or <p>. The most important attribute of the <a> tag is the href, which specifies the URL the link points to. It can also include other attributes like target, which defines where the new page will be displayed, and rel, which specifies the relationship between the linked page and the current page.

Here's a simple example: Visit onlinelectureshub.com. This will create a link that, when clicked, opens onlinelectureshub.com in a new browser tab.

By default, links appear as:

  • An unvisted link is underlined and blue
  • A visted link is underlined and purple
  • An active link is underlined and red

A TAG Attributes

Attribute What it does
target Specifies where to open the linked document
href Specifies URL of the page the link goes to
hreflang Specifies the language of the linked document
type Specifies the media type of linked document
download Target will be downloaded when user clicks on the hyperlink
ping Target will be pinged when user clicks on the hyperlink
referrerpolicy Specifies the referrer policy of the linked document
rel Specifies the relationship between the current document and linked document
media Specifies the media type of linked document
