Table Tag

  1. Tags

    2. The table tag is used to create a table with rows and columns to organize data. There are 3 tags that are used when creating a table. These tags include the <tr>, <th>, and <td> tags. The <tr> tag is the table row tag which is used to define a row within a table. The <th> tag is the table header cell. This tag is used to create table header cells and define them. The <td> tag is the table data cell which is used to define data cells in a table.

  2. Attributes

    3. The Table tag has many attributes including border, cellpadding, cellspacing, width, height, align, and valign. The border attribute specifies the width of a table's border. The cellpadding attribute defines the space between a cell's content and its borders. The cellspacing attribute defines the amount of space between cells in the table. The width and height tags set the width and heights of the table. Finally, align and valign align the table horizontally and vertically. The align attribute has 3 settings: left, center, and right. The valign attribute also has 3 settings: top, middle, and bottom. The Table tag also includes the colspan and rowspan attributes. The colspan attribute is used to specify the number of columns a cell will span horizontally. On the other hand, the rowspan attribute is used to specify the number of rows a cell will span vertically.

Tag Description
<table> Creates a table
<tr> Defines a row inside a table
<th> Defines header cells in the table
<td> Defines regular data cells in the table

Attribute Description
border Specifies border width
cellpadding Specifies space between cell content and borders
cellspacing Specifies space between cells
width Sets table width
height Sets table height
align Aligns table horizontally using left, center, or right
valign Aligns table vertically using top, middle, or bottom
rowspan Used to make a cell span multiple rows
colspan Used to make a cell span multiple columns
